# CollegeJournalApp

Приложение для ведения журнала колледжа.

## Функционал

- Управление студентами, группами и пользователями
- Учёт посещаемости и оценок
- Просмотр расписания занятий
- **Импорт расписания из Excel**

## Импорт расписания из Excel

Для загрузки расписания из файла Excel:

1. Откройте страницу **"Расписание"** в приложении.
2. Нажмите кнопку **"📥 Загрузить из Excel"**.
3. Выберите файл формата `.xlsx` или `.xls`.

### Формат файла Excel

Файл должен содержать следующие столбцы (в порядке следования):

| № | Столбец | Описание | Пример |
|---|---------|----------|--------|
| 1 | День недели | Название дня на русском | Понедельник, Вторник, ..., Суббота |
| 2 | Номер урока | Порядковый номер занятия | 1, 2, 3 |
| 3 | Время | Диапазон времени в формате "HH:MM – HH:MM" | 08:30 – 10:00 |
| 4 | Дисциплина | Название предмета | Математика, Физика |
| 5 | Аудитория | Номер кабинета | 101, 205а |
| 6 | Преподаватель | ФИО преподавателя | Иванов И.И. |
| 7 | Тип недели | "Чётная", "Нечётная" или "Обе" | Обе |

**Первая строка считается заголовком и пропускается.**

#### Пример данных в Excel:

| День недели | № | Время | Дисциплина | Аудитория | Преподаватель | Тип нед. |
|-------------|---|-------|------------|-----------|---------------|----------|
| Понедельник | 1 | 08:30 – 10:00 | Математика | 101 | Петров П.П. | Обе |
| Понедельник | 2 | 10:15 – 11:45 | Физика | 205 | Сидоров С.С. | Чётная |
| Вторник | 1 | 08:30 – 10:00 | Химия | 301 | Иванов И.И. | Нечётная |

## Требования

- .NET Framework 4.8
- Библиотеки:
  - ClosedXML (для работы с Excel)
  - Microsoft.Data.SqlClient (для подключения к БД)
  - BCrypt.Net-Next (для хеширования паролей)

## Установка

1. Клонируйте репозиторий.
2. Откройте решение в Visual Studio.
3. Восстановите NuGet-пакеты.
4. Настройте подключение к базе данных в `App.config`.
5. Соберите и запустите проект.
